# meta-mc

> 当前版本 `2.8.4`

### 使用方法

下载后替换至`miao-plugin/resources/meta-mc`文件夹

此库更新比liangshi-calc的自动更新晚1~16小时

推荐使用liangshi-calc的自动更新，仅在网络不佳无法连接至API时使用此库。

### 其他说明

此库不含计算/评分等其他功能，仅含必要数据

图片和数据仅供交流学习使用，内容均来源于网络。

> 下次更新时间(预估) 2025-11-14 00:00(UTC+8)
