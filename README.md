# meta-mc

> 当前版本 `3.1.4`

## 使用方法

### 安装方法

#### 手动替换

下载ZIP后解压并替换至`miao-plugin/resources/meta-mc`文件夹即可使用

#### 手动下载

在根目录下执行

~~~
git clone --depth=1 https://gitee.com/liangshi233/meta-mc.git ./plugins/miao-plugin/resources/meta-mc/
~~~

即可使用

#### 自动下载

在安装了`liangshi-calc`的bot中使用指令

~~~
#一键更新鸣潮新版本完整数据
~~~

即可使用

### 更新方法

#### 手动替换

下载ZIP后解压并替换至`miao-plugin/resources/meta-mc`文件夹进行更新

#### 手动更新

手动在`plugins/miao-plugin/resources/meta-mc`内运行

~~~
git pull
~~~

#### 自动更新

在安装了`liangshi-calc`的bot中使用指令

~~~
#一键更新鸣潮新版本数据
~~~

---

推荐使用liangshi-calc的自动更新，仅在网络不佳无法连接至API时使用此库。

## 其他说明

此库不含计算/评分等其他功能，仅含图鉴与其他基础功能的必要数据

图片和数据仅供交流学习使用，内容均来源于网络。
